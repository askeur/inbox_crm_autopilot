# inboxCRM-autopilot
Autopilot CRM — Pipeline IA à 5 agents pour traiter vos emails automatiquement  Chaque email reçu est classifié, analysé, enregistré en base, suivi d'une réponse rédigée et d'une relance planifiée — le tout en moins de 3 secondes.  Stack : Groq (llama-3.3-70b) · FastAPI · Streamlit · PostgreSQL · Docker
